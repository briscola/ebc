### 料金計算のテスト方法について

いくつかの電力会社は料金計算サービスを提供しており、それを使ってテストを行うことができる。

[東京電力の料金計算サービス](https://www2.kakeibo.tepco.co.jp/ratesim/)

[北海道電力の料金計算サービス](http://www.hepco.co.jp/userate/price/simulation/)

[東北電力の料金計算サービス](http://www.tohoku-epco.co.jp/denki/simulation/)

[北陸電力の料金計算サービス](http://www.rikuden.co.jp/shisan/)

[中部電力の料金計算サービス](http://www.chuden.co.jp/home/shokai/simulation2/index.html)

[関西電力の料金計算サービス](http://www.kepco.co.jp/home/ryoukin/simu/index.html)

[中国電力の料金計算サービス](http://www5.energia.co.jp/EIGYO/HTML/LW1W1000.htm)

[四国電力の料金計算サービス](http://www.yonden.co.jp/cgi-bin/ryokin/menu.cgi)

[九州電力の料金計算サービス](http://www.kyuden.co.jp/rate_simm_index.html)

[沖縄電力の料金計算サービス](http://www.okiden.co.jp/service/calculation/calc_assumption_index.html)
